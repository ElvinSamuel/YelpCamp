#MONGODB Commands

##CRUD Operations: Create, Read, Update & Destroy
### mongod - the mongo daemon, starts the process. Exists in a separate terminal.
### mongo - starts the Mongo CLI
### help - shows a list of commands for the Mongo Shell
### show dbs - shows all databases
### use <database> - switches to specified database <database>
### insert - add data to a collection
### find - (along with 'pretty()') show data in a collection
### update - edit data in a collection.
### remove - delete data from a collection.
